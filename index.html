Skeleton Slayer groep project

Dit project heb ik ervoor gekozen alles voor de rooms gedaan: hoe die eruit zien, de random generation, player en enemy detection. Aangezien ik veel roguelikes speel wilde ik me hier in verdiepen. De random generation heb ik zelf uitgewerkt (zonder tutorial) en wordt gemanaged met een singleton die alle prefabs in arrays heeft staan.

Cube Souls


Dit was een solo project waar ik 4 weken aan heb gewerkt. Voor dit project heb ik ook zelf de models gemaakt van alles in de game. Deze zijn gemaakt in MagicaVoxel.

Dit is een wave fighting game. De combat is gebaseerd op Dark Souls. Github link: https://github.com/DeLaBlok/Project-voxel.
